# Dungeon-crawler-style-RPG

## 📌 概要
Javaを用いて開発した、テキストベースで進行するダンジョン探索型RPGです。  
プレイヤーがダンジョンを探索し、敵との戦闘や成長を通じて進行していくゲームを実装しています。

本プロジェクトでは、ゲーム開発を通じて**オブジェクト指向設計とロジック構築の理解**を深めることを目的としました。

---

## 🎮 ゲーム内容

- プレイヤーがダンジョンを探索
- 敵との戦闘システム
- HPや攻撃力などのステータス管理
- 勝敗による進行・終了判定

---

## 🧠 工夫した点

### ① オブジェクト指向設計の活用
プレイヤー・敵・ゲーム進行などをクラスとして分離し、それぞれの責務を明確にしました。  
これにより、拡張しやすく保守性の高い構成を意識しています。

---

### ② 戦闘ロジックの実装
攻撃・ダメージ計算・HP管理などのゲームロジックを実装し、  
ゲームとして成立する基本的な仕組みを構築しました。

---

### ③ テキストベースでのゲーム設計
GUIではなくCUI（コマンドライン）で動作するように設計し、  
ロジック部分の理解に集中できる構成としました。

---

## 技術スタック

- Java
- 標準入出力（CUIベース）
